# License Key System / API (FiveM Dev Template)

This project is a remake of an older script: a License Key System / API designed for FiveM developers. It acts as a **template** for developers or communities who want to **lock products behind license keys**.

## Overview

- All license management is done through a **Discord bot**.  
- The bot stores key information in a **MySQL database**.  
- It automatically sets up **ports and endpoints** on the network where it is running.  
- Originally written in JavaScript, this new version is being rewritten in **TypeScript** for better structure and maintainability.  

## Features

- Generate, validate, and revoke license keys through Discord commands.  
- Seamless integration with your FiveM development projects.  
- Network endpoints are automatically handled for your API needs.  
- Fully modular and intended as a **developer template**, so you can extend or modify it to your needs.  

## Future Plans

- A **web panel** may be added in the future to provide a graphical interface for managing licenses, but this is not available yet.  
- Additional enhancements for scalability and ease-of-use are possible, depending on community feedback.  

## Getting Started

1. Set up a **MySQL database** for storing license information.  
2. Configure the Discord bot with your server and API credentials.  
3. Run the bot to automatically initialize network endpoints and start managing keys.  

## Usage

- Ideal for developers running a **FiveM development community**.  
- Use it as a base for secure license management of scripts, mods, or other digital products.
